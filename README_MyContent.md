# Client Management App

##  Description
A React application to manage clients with CRUD operations.
Built using modern React concepts.

##  Tech Stack
- React
- Redux Toolkit
- React Router DOM
- Styled Components

## Features
- Add client
- Edit client
- Delete client
- State management with Redux
- Routing with React Router

## Concepts Used
- Components architecture
- Redux Toolkit slices
- Async actions
- Props & state
- React Router

npm install
npm run dev

npm install styled-components
